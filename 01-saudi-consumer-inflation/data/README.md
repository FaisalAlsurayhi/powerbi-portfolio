# Data Notes

This folder contains sample data for the Saudi Consumer Inflation Dashboard.

The file `saudi_consumer_inflation_sample.csv` is synthetic portfolio data. It was created for dashboard practice and should not be treated as official economic data.

## Columns

- `Month`: Reporting month
- `Region`: Saudi region group used in the dashboard
- `Category`: Consumer or retail category
- `CPI_Index`: Sample consumer price index value
- `Inflation_Rate`: Sample inflation rate percentage
- `Household_Spending_SAR_M`: Sample household spending in SAR millions
- `Retail_Sales_SAR_M`: Sample retail sales in SAR millions
- `Purchasing_Power_Index`: Sample index showing relative purchasing power

## Use

Use this dataset to rebuild or refresh the Power BI report if the original source file is not available.
