# Power BI Portfolio

This repository contains my Power BI dashboard projects. I use these projects to practice data cleaning, data modeling, DAX measures, and dashboard storytelling.

The goal is to show how I approach a business question, build a simple reporting structure, and turn raw data into visuals that are easier to understand for a manager or stakeholder.

This repo is part of my broader data analytics portfolio, which mostly focuses on Saudi and Eastern Province business topics.

## Tools Used

- Power BI
- Excel / CSV files
- DAX
- Basic data cleaning and dashboard design

## Projects

| Project | Description |
| --- | --- |
| [Saudi Consumer Inflation Dashboard](01-saudi-consumer-inflation/) | A Power BI dashboard looking at inflation, household purchasing power, consumer spending, and retail trends in Saudi Arabia. |
| [Eastern Province Real Estate Market Dashboard](02-eastern-province-real-estate-dashboard/) | A Power BI dashboard analyzing sales value, transaction activity, pricing, and property mix across Eastern Province cities. |

## What I Try To Show In Each Dashboard

- A clear business question before building visuals
- KPI cards that summarize the main story quickly
- Pages that move from overview to deeper analysis
- Simple DAX measures for totals, averages, and trend comparison
- Screenshots and notes so the project can be understood without opening Power BI
- Honest limitations when the data is synthetic or simplified

## Skills Demonstrated

- Building interactive Power BI dashboards
- Creating report pages for different analysis angles
- Working with Excel/CSV-style datasets
- Writing DAX measures for calculations and KPIs
- Defining business questions before choosing visuals
- Explaining dashboard findings in plain language
- Organizing a Power BI project for GitHub
- Using screenshots and documentation to explain dashboard work
