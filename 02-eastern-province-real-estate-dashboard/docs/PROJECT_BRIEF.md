# Eastern Province Real Estate Market Dashboard

## Project Overview

This Power BI project analyzes a synthetic real estate transaction dataset for major Eastern Province cities:

- Khobar
- Dammam
- Dhahran
- Jubail

The dashboard is designed as a business reporting project, not a political or policy project. It focuses on market activity, city comparison, property type performance, and district-level insights.

## Business Questions

- Which cities have the highest transaction volume?
- Which cities have the highest total sales value?
- Which property types drive the most activity?
- Which districts have higher average price per square meter?
- How does residential activity compare with commercial activity?
- Which months show stronger market activity?

## Dataset Notes

The dataset is synthetic and created for portfolio practice. It is designed to look realistic enough for a Power BI dashboard, but it should not be treated as official market data.

## Dashboard Pages

1. **Market Overview**
   - Overall KPIs and monthly trend
2. **City Comparison**
   - City-level activity and price comparison
3. **Property Type Analysis**
   - Residential and commercial property mix
4. **District Insights**
   - District-level value and price analysis

## Main KPIs

- Total Transactions
- Total Sales Value
- Average Sale Price
- Average Price per sqm
- Total Area Sold
- Residential Share
- Commercial Share
